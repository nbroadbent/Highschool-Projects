How To Use:

Go to: www.questrade.com and log in:	https://practicelogin.questrade.com/Signin.aspx?ReturnUrl=%2fAPIAccess%2fuserapps.aspx

Click "TAKE ME TO THE APP HUB"

Click: "Register a personal app"

Fill out the form. Make sure to select each of the three checkboxes, giving the trader full access. Ignore the callback urls.

Next, click: "+ New device" on the right.

Click: "Generate new token" on the right. 

Copy the token and use it in the AlgoTrader program.
